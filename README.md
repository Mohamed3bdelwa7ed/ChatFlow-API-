# ChatFlow-API-

A simple AI-powered chat application built with **.NET Console App** 
that connects to multiple AI models through the **AgentRouter API gateway**.

## ✨ Features

- Real-time AI chat in the terminal
- Supports multiple AI models (Claude, DeepSeek, GPT, etc.)
- Clean HTTP integration using `HttpClient`
- JSON serialization with `System.Text.Json`

## 🚀 Getting Started

### Prerequisites
- .NET SDK 6.0 or later
- An AgentRouter API Key → [agentrouter.org](https://agentrouter.org)

### Installation

```bash
git clone https://github.com/Mohamed3bdelwa7ed/ChatFlow-API.git
cd AgentRouter-AI-Chat
```

Open `Program.cs` and replace the API key:
```csharp
var apiKey = "YOUR_AGENT_ROUTER_KEY";
```

Then run:
```bash
dotnet run
```
